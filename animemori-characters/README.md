# Animemori Characters v0.1.1

This plugin adds automatic character importing using a public API endpoint.

## Requirements
- Animemori Core active (database tables come from the core plugin).
- WordPress 6.0+
- PHP 8.0+

## Install
1) Upload the ZIP in WP Admin -> Plugins -> Add New -> Upload Plugin
2) Activate
3) Go to WP Admin -> Animemori -> Characters

## Notes
- If Animemori Core version is 0.3.0 or newer, Characters are already included in the core and this plugin will deactivate itself.
